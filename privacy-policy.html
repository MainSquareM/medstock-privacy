📋 Política de Privacidade - MedStock

Última atualização: 1 de Março de 2026

🔒 PRIVACIDADE TOTAL: O MedStock não envia NENHUM dado para servidores externos. Tudo fica armazenado exclusivamente no seu dispositivo. Não partilhamos, vendemos ou recolhemos informações pessoais.

1. INTRODUÇÃO

O MedStock é uma aplicação de gestão de medicamentos domésticos desenvolvida por MainSquare (geral@mainsquare.pt) com o compromisso de proteger a sua privacidade. Esta política descreve como tratamos as suas informações e está em conformidade com o Regulamento Geral sobre a Proteção de Dados (RGPD/GDPR) da União Europeia.

2. DADOS RECOLHIDOS

O MedStock recolhe e armazena LOCALMENTE no seu dispositivo:

  • • Nomes de medicamentos
  • • Informações de stock (quantidade, validade)
  • • Horários de toma e posologia
  • • Fotografias de embalagens (se fornecidas)
  • • Preferências da aplicação

3. ARMAZENAMENTO DE DADOS

⚠️ IMPORTANTE: Todos os dados são armazenados EXCLUSIVAMENTE no seu dispositivo

  • • NÃO enviamos dados para servidores externos
  • • NÃO partilhamos informações com terceiros
  • • NÃO recolhemos dados de utilização ou analytics
  • • NÃO temos acesso aos seus dados

4. NOTIFICAÇÕES

  • • As notificações são processadas localmente no dispositivo
  • • Não temos acesso ao conteúdo das notificações
  • • Pode desativar notificações a qualquer momento

5. FOTOGRAFIAS, CÂMARA E SCANNER

  • • O acesso à câmara é usado apenas para fotografar embalagens e ler códigos de barras de medicamentos
  • • As fotografias são armazenadas apenas localmente
  • • Os dados dos códigos de barras são processados apenas localmente
  • • Não são enviados para nenhum servidor

6. CÓPIAS DE SEGURANÇA (BACKUPS)

  • • Os backups são criados e armazenados no seu dispositivo
  • • Tem controlo total sobre a exportação/importação de dados
  • • Backups partilhados são da sua responsabilidade

7. ELIMINAÇÃO DE DADOS

  • • Pode eliminar todos os dados a qualquer momento
  • • Desinstalar a aplicação remove todos os dados
  • • Não mantemos cópias dos seus dados

8. SEGURANÇA

  • • Os dados são protegidos pelo sistema de segurança do seu dispositivo
  • • As cópias de segurança são encriptadas para proteção adicional
  • • Recomendamos usar bloqueio de ecrã no dispositivo

9. CRIANÇAS

  • • A aplicação não é direcionada a menores de 13 anos
  • • Dados de menores devem ser geridos por adultos responsáveis

10. BASE LEGAL E DIREITOS (RGPD)

A base legal para o tratamento dos seus dados é o consentimento e o interesse legítimo (funcionamento da aplicação). Uma vez que todos os dados são armazenados localmente no seu dispositivo, o utilizador tem controlo total. Nos termos do RGPD, tem direito a:

  • • Acesso — todos os dados são visíveis diretamente na aplicação
  • • Retificação — pode editar qualquer dado a qualquer momento
  • • Eliminação — pode apagar qualquer dado ou desinstalar a app
  • • Portabilidade — pode exportar os seus dados através da funcionalidade de backup

11. ALTERAÇÕES À POLÍTICA

  • • Quaisquer alterações serão comunicadas através de atualizações da app
  • • Continuar a usar a app após alterações implica aceitação

12. CONSENTIMENTO

Ao utilizar o MedStock, concorda com esta Política de Privacidade.
